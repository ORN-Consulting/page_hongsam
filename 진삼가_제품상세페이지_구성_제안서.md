# 진삼가 제품 상세페이지 구성 제안서

### 브랜드 스토리 × 제품 정보 × 구매 전환 통합 설계

---

<br>

## 설계 철학

> **"제품을 설명하는 페이지"가 아니라, "브랜드 스토리를 경험하다가 자연스럽게 구매하게 되는 페이지"**

기존 진삼가 제품 상세페이지는 제품 스펙(규격, 가격)만 나열하고, 브랜드의 핵심 자산인 9증9포 기술, 풍기 6년근 원료, 합성첨가물 제로라는 스토리가 구매 흐름에 연결되지 않았습니다. 이 제안서는 고객이 페이지를 스크롤하는 동안 자연스럽게 "왜 진삼가인가?"를 납득하고, 마지막에 확신을 가지고 구매 버튼을 누르는 구조를 설계합니다.

<br>

## 디자인 기본 원칙

| 항목 | 규칙 |
|---|---|
| **타이틀 폰트** | Gowun Batang (Bold 700 / Regular 400) — 전통·정성·자연의 온기 |
| **본문 폰트** | Gowun Dodum (Regular 400) — 부드러운 가독성 |
| **브랜드 컬러** | 딥레드 #8B0000 (타이틀·강조) / 골드 #B8860B (서브 악센트) / 다크그레이 #333 (본문) |
| **배경 톤** | 따뜻한 아이보리 #F5F5F0 (섹션 배경 교차) / 화이트 #FFFFFF |
| **이미지 스타일** | 따뜻한 자연광 톤, 소백산·풍기 연상 자연 배경, 크래프트 느낌 |
| **전체 분위기** | 프리미엄이되 차갑지 않고, 전통적이되 세련된 — "손으로 빚은 고급" |

### Section 1. 제품 기본 정보 (Above the Fold)

**고객 심리:** "이게 뭐지? 내가 찾는 건가?"

**목적:** 스크롤 없이 핵심 정보를 모두 파악하고 구매 가능하게

- 우측 40%: 제품 정보. 가장 위에 "9증9포 명품홍삼" 서브 라벨(Gowun Dodum, 13px, 골드), 그 아래 제품명(Gowun Batang Bold, 28px), 한 줄 설명(Gowun Dodum, 16px)
- 3대 차별점 뱃지를 시각적 박스로 강조 — 이 영역이 고객이 가장 먼저 읽는 "왜 이 제품인가?"에 대한 답
- 가격 → 옵션 → 수량 → CTA 버튼 순서. 두 개 버튼(장바구니/바로구매)을 병렬 배치

### Section 2. 브랜드 한 줄 연결 (Story Bridge)

**고객 심리:** "좋아 보이는데... 왜 이렇게 비싸지?"

**목적:** 스크롤을 유도하며, 아래 섹션에서 "이유"를 보여줄 것임을 예고

```
┌─────────────────────────────────────────────────────────────┐
│                     [따뜻한 아이보리 배경]                     │
│                                                             │
│        [Gowun Batang Regular, 22px, 딥레드]                  │
│        "우리는 어떤 일을 하는가?"                              │
│                                                             │
│        [Gowun Dodum, 15px, 다크그레이]                       │
│        우리는 사람들의 건강을 지키고                            │
│        행복한 삶을 살아가는 데 도움을 줍니다.                    │
│                                                             │
│                    — 진삼가 대표 김명범                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**상세 설명:**

- 브랜드 스토리의 철학 문구를 간결하게 인용. 여기서 전체 브랜드 스토리를 장황하게 풀지 않고, "철학이 있는 회사"라는 인상만 심음
- 배경은 따뜻한 아이보리(#F5F5F0)로 전환하여 섹션 구분 + 온기 있는 분위기 조성
- 모바일에서는 전체 화면 한 블록으로 보이는 호흡 구간 역할

### Section 4. 9증9포 기술 — 공정의 진정성

**고객 심리:** "9증9포가 뭔데? 일반 홍삼이랑 뭐가 다른 거지?"

**목적:** 가격 차이의 핵심 이유를 시각적 비교로 납득시킴

```
│  │                                      │                    │
│  │  수삼 → 1차증숙 → 1차건조 → ...       │                    │
│  │        → 9차증숙 → 9차건조 → 홍삼     │                    │
│  │                                      │                    │
│  │  0.1도 단위 정밀 온도 제어             │                    │
│  │  3~4일간 자동 반복                    │                    │
│  │                                      │                    │
│  └──────────────────────────────────────┘                    │
│                                                             │

│  │                                              │           │

│  │                                              │           │
│  │  ┌──────────────┬──────────┬──────────┐      │           │
│  │  │              │ 9증9포   │ 일반1증1포│      │           │
│  │  ├──────────────┼──────────┼──────────┤      │           │
│  │  │ 진세노사이드  │ 5~12배   │ 기준치   │      │           │
│  │  │ 사포닌 손실   │ 거의 없음 │ 손실 발생│      │           │
```

- 하단: 인증 뱃지 + 김명범 대표의 김치찌개 비유 인용
- 이 섹션이 **가격 정당화의 핵심**. "360,000원이 비싼 게 아니라, 5~12배의 진세노사이드를 담는 기술에 대한 대가"임을 납득시킴

### Section 5. 합성첨가물 제로 — 성분의 진정성

**고객 심리:** "진짜 좋은 건 알겠는데, 다른 건 안 들어있지?"

```
│  │  ✓ 홍삼추출액 100%                           │            │
│  │    (풍기산 6년근 인삼, 9증9포 방식 추출)        │            │
```
