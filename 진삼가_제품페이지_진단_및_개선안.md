| 1-5 | **제품 라인업 체계 부재** | 🟡 중간 | 누구를 위한 제품인지(타깃), 어떤 상황에 적합한지(용도)에 대한 안내 없이 기능적 스펙만 나열 |

### 2. 제품 상세 페이지 (개별 제품) 개선안

제품 상세페이지는 아래 순서로 구성하는 것을 권장합니다. 각 섹션별로 역할과 이유를 함께 설명합니다.

#### 섹션 1: 제품 기본 정보 (Above the Fold)

```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│  [제품 이미지 갤러리]          [제품 정보 영역]            │
│   - 메인 이미지                 진삼가 진 (眞)            │
│   - 360도 뷰 or 썸네일들        프리미엄 홍삼추출액        │
│   - 패키지 + 내용물 사진         ─────────────────        │
│                                 ★★★★★ (리뷰 수)         │
│                                 360,000원                │
│                                 ─────────────────        │
│                                 · 홍삼추출액 100%         │
│                                 · 진세노사이드 12배        │
│                                 · 합성첨가물 제로          │
│                                 · 80ml × 30포/100포       │
│                                 ─────────────────        │
│                                 옵션: [30포 ▼] [100포]    │
```

```
─── 9증9포, 9번의 정성이 만든 차이 ───

[인포그래픽: 1증1포 vs 9증9포 비교]

· 일반 홍삼 대비 진세노사이드 5~12배
· 0.1도 단위 전자동 정밀 온도 제어
· 13종 특허 기술

[인포그래픽: 잔뿌리 비교 사진 — 1증1포 vs 9증9포]
```

**목적:** 가격 정당화. "왜 비싼지"가 아니라 "왜 이 가격이 합리적인지" 설득

## Part 3. 핵심 요약

### 현재 페이지의 근본 문제

> **"우리 제품을 보여준다"는 판매자 관점으로 설계됨.**
> 고객은 "내가 뭘 사야 하는지" 알 수 없는 상태.

### 개선의 핵심 방향

> **"당신에게 맞는 제품을 찾아드립니다"는 고객 관점으로 전환.**
> 모든 정보 설계를 "구매 결정을 돕는 것"에 집중.

### 가장 시급한 3가지 (비용 대비 효과 최대)

1. **제품 카드에 "진세노사이드 ○배"와 한 줄 차별점 추가** — 코드 수정만으로 즉시 가능
2. **목적별 제품 분류 탭 (나를 위한 / 선물용 / 자녀용 / 간편 휴대)** — 고객 선택 장애 해소
3. **중복 제품 제거 + 5+1 묶음을 단품 상세 내 옵션으로 통합** — 페이지 정리

이 세 가지만 실행해도 제품 페이지의 사용자 경험과 전환율이 크게 개선될 것입니다.
